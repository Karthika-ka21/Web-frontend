Largest of Three Numbers

let x=Number(prompt("enter the first number:"));
let y=Number(prompt("enter the second number:"));
let z=Number(prompt("enter the third number:"));
let result=(x>y)?((x>z)?x:z):((y>z)?y:z);
console.log(`${result} is the largest number`);

Note:we must convert the prompt input to Number using Number() function because by default prompt takes input as string.
Because of this if we don't convert it to number the comparison will be done on the basis of string which may give wrong results.
